## New Features in Standalone Mode

### 1. Recipes 📋
**What it is:** Pre-built workflows for common development tasks

**Available recipes:**
- 🔍 Code Review - Analyze code quality
- 🐛 Bug Fix - Identify and fix bugs
- ♻️ Refactor Code - Improve maintainability
- 🧪 Generate Tests - Create test suites
- 📝 Write Documentation - Generate docs
- ✨ Feature Scaffold - Create boilerplate

**How it works:**
- Click any recipe card
- Recipe runs in chat context
- Results shown in chat

**Note:** Advanced multi-file recipes require Server Mode

### 2. Extensions 🔌
**What it is:** MCP-compatible tools and integrations

**Available extensions:**
- ✅ Filesystem (enabled by default)
- 🔌 GitHub (can be enabled)
- 🔍 Web Search (can be enabled)
- 🗄️ Database (can be enabled)

**How it works:**
- Extensions provide tools to AI
- Filesystem always enabled
- Others can be toggled
- Custom extensions in Server Mode

**Workspace:**

- ❌ Kanban board (needs database)

- ❌ Apps (needs hosting)
- ❌ Multi-agent orchestration (needs coordination)
- ❌ Release trains (needs team features)
- ❌ WorkerBees (needs isolation)
- ❌ Advanced recipes (needs multi-agent)
- ❌ Custom extension management
