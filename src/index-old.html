🌐 Switch to Server Mode

Server mode gives you multi-agent orchestration, team collaboration, and advanced features.

⚠️ Requirements: Docker Desktop must be installed to run the Squan server.

Step 1: Install Docker Desktop

Download and install Docker Desktop for your platform:

Download Docker Desktop →

After installing, restart your computer and launch Docker Desktop.

Step 2: Download Squan Server Files

Open PowerShell (Windows) or Terminal (Mac/Linux) and run:

Windows (PowerShell):
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/tarvitave/squan-release/main/start-windows.ps1" -OutFile "start.ps1"; .\start.ps1
macOS / Linux:
curl -O https://raw.githubusercontent.com/tarvitave/squan-release/main/start-mac-linux.sh && chmod +x start-mac-linux.sh && ./start-mac-linux.sh

Step 3: Configure API Keys

The setup script will prompt you to add at least one API key. You can also edit the .env file manually:

ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GOOGLE_API_KEY=...

Step 4: Start Squan Server

Once Docker is running and your API keys are configured:

docker compose up -d

The server will start on http://localhost:3000

Once the server is running, click the button below to connect:

About Squan

Version 2.0.2

Squan is a universal AI development platform that lets you orchestrate multiple AI agents for complex software development tasks.

Standalone Mode (Current)

Chat directly with AI models without a server. Perfect for solo developers who want simplicity and speed.

  • ✅ No Docker required
  • ✅ 3 AI providers (Claude, OpenAI, Gemini)
  • ✅ Real-time cost tracking
  • ✅ Settings saved locally

Server Mode (Optional)

Connect to a Squan server for multi-agent orchestration, team collaboration, and advanced features.

  • ✅ Multiple parallel agents
  • ✅ Root Agent + WorkerBees
  • ✅ Team collaboration
  • ✅ Advanced workflows
