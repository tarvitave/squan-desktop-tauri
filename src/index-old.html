1 Start Squan with Docker

The fastest way to get Squan running locally:

# Download docker-compose.yml curl -O https://raw.githubusercontent.com/tarvitave/squan-release/main/docker-compose.yml # Download .env.example curl -O https://raw.githubusercontent.com/tarvitave/squan-release/main/.env.example # Rename and configure mv .env.example .env # Edit .env with your API keys # Start Squan docker-compose up -d # Access at http://localhost:3000
